# 🔍 NumbScan – Network Device & Port Scanner

**NumbScan** is a lightweight Python tool for scanning devices on your local network and optionally checking for open TCP ports. Built using `scapy` and `socket`, it's a simple yet powerful utility for network engineers, ethical hackers, or anyone curious about what’s on their LAN.

---

## 🚀 Features

- ⚡ Fast ARP-based device discovery (IP + MAC address)
- 🌐 Auto-detects your local subnet
- 🐍 Pure Python – no external dependencies except `scapy`

---

## 📦 Requirements

- Python 3.6+
- `scapy` module
